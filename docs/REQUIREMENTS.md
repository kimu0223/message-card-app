# 要件定義

## プロダクト概要
アニメーション付きメッセージカードWebアプリ。
「デザインセンスがない男性でも3分で感動的なカードが作れる」を最優先コンセプトとする。

## ターゲットユーザー
- 主: 20〜40代男性（デザイン経験なし）
- 副: メッセージカードを頻繁に送る全ユーザー

## コアユーザーフロー（最短ルート）
1. テンプレート選択（シチュエーション → スタイルの2ステップ）
2. AIメッセージ生成（自動表示・ワンクリック適用）
3. 微調整（テキスト・色・フォントのみ）
4. アニメーション付きプレビュー確認
5. シェアURL発行 / PDF印刷

## 機能要件

### Phase 1 MVP（必須）
- テンプレート選択・カード作成・保存
- テキスト編集・背景色変更・フォント選択
- AIメッセージ生成（Gemini）
- シェアURL（公開閲覧ページ）
- PNG/PDF書き出し
- Google OAuth認証

### Phase 2
- アニメーション（confetti・fade・slide・bounce・sparkle）
- カードダッシュボード（お気に入り・削除）

### Phase 3
- Stripe課金（Proサブスク + クレジット購入）
- プランゲーティング

### Phase 4（将来）
- AIデザイン提案・AI画像生成
- LINE/Instagram OGP最適化
- モバイルUI最適化

## 非機能要件
- LCP ≤ 2.5s（Core Web Vitals）
- モバイル対応（ビューアは完全対応、エディタはタブレット以上を推奨）
- SEO: OGP対応（シェアカードページ）
