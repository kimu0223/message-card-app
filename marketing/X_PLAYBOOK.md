# X（Twitter）戦略プレイブック — 贈りことば

---

## 基本戦略

**ポジション**: 「プロっぽいカードが自分でも作れる」ことを体験させる

**Xでの役割**: ファネルのTOFU。「知ってる人が使いたくなる」コンテンツに徹する。

**NGコンテンツ**: 機能紹介、料金説明、会社の話

---

## 投稿タイプ別テンプレート

### TYPE A: 完成カード動画（週2本）

**目的**: 完成物のクオリティを見せる。「これ作りたい」を引き出す

**構成**:
```
[動くカードのGIF/動画 15秒]

父の日に、言葉で伝えてみませんか。

アニメーション付きのカードが3分で作れます。
登録不要・無料で試せます。

→ [リンク: /create?template=father&utm_source=x&utm_content=video_a]
```

**制作指針**:
- 動画は15秒以内（X自動再生の視聴完了率最大化）
- 音なしでも伝わる（字幕 or テキストオーバーレイ）
- カードのアニメーションを最初の2秒以内に見せる

---

### TYPE B: シーズン訴求（イベント2週前から）

**目的**: 締め切り感で行動を促す

**構成**:
```
父の日まで、あと14日。

「ありがとう」って、
言えないままにしてませんか？

3分で作れる、アニメーション付きメッセージカード。
喜ばれること、保証します。

→ [リンク]
```

| ハロウィン | 2026-10-31 | 10-17 |

### TYPE C: ビフォーアフター（週1本）

**目的**: 「自分でもできる」を証明する

**構成**:
```
LINEで「おめでとう」って送るだけより、

こんなカードが届いたら──────

[完成カードのスクリーンショット or 動画]

相手の顔、想像してみてください。

テンプレートを選ぶだけ。文字を入れるだけ。
それだけで、こういうのが作れます。

→ 無料で試す: [リンク]
```

### TYPE D: How-to Thread（週1スレッド）

**目的**: 「使い方がわかれば使える」という人向け。SEO的効果も期待

**5ポスト構成**:
```
1/ 「デザインの才能がなくても、こういうカードが作れます」
[完成動画]
作り方、一緒に見てください👇

2/ ステップ1: テンプレートを選ぶ（30秒）
[テンプレート選択画面のスクリーンショット]
誕生日・記念日・お礼など、シーン別に選べます

3/ ステップ2: メッセージを入力（1分）
[メッセージ入力画面]
AIが文章を提案してくれます
「何を書けばいいかわからない」も解決

4/ ステップ3: 送る（30秒）
[シェア画面]
URLをコピーしてLINEやメールで送るだけ
相手はアプリなしで開けます

5/ 以上、3分でこんなカードが作れます
[完成カード動画]

登録不要、無料で試せます
→ [リンク]
```

## リンク設計

X投稿のリンクは必ず UTM パラメータを付与してGA4でトラッキング:

```
# 基本形
https://okurikotoba.com/create?utm_source=x&utm_medium=organic&utm_campaign=[campaign]&utm_content=[type]

# テンプレート直リンク（カード動画投稿時）
https://okurikotoba.com/create?template=[id]&utm_source=x&utm_content=video

# シーズン専用
https://okurikotoba.com/create?occasion=father&utm_source=x&utm_campaign=fathers_day
```

## 反応別フォローアップ

| ユーザーアクション | 対応 |
|---|---|
| 「どこで作ったの？」リプライ | 即リプ: 「こちらです！無料で使えます→[リンク]」 |
| 完成カードのRT | 「素敵に使ってもらえて嬉しいです！」でエンゲージ |
| 「使ってみた」報告 | 引用RT + コメント → バイラル促進 |
| 否定的なコメント | 真摯に返信。改善につなげる |

## コンテンツ素材の作り方

### アニメーションGIF作成手順
1. `贈りことば` エディタでカードを完成させる
2. プレビュー画面でブラウザ録画ツール（OBS or Loom等）で15秒録画
3. GIF変換（gifski推奨）
4. Xにアップロード（3MB以内推奨）

### スクリーンショット撮影
- ブラウザ幅: 1280px（デスクトップ）または 390px（iPhone15相当）
- 余分なUIは除去してカードを大きく見せる
